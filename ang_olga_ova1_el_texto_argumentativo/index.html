El texto argumentativo

Introducción

Este objeto virtual de aprendizaje, en primer momento, presenta una imagen, acompañada de preguntas orientadoras que tienen como finalidad activar los conocimientos previos sobre el texto argumentativo. Seguidamente, se comparte información sobre la estructura, características, propósito comunicativo y ejemplos del texto argumentativo; como retroalimentación se invita a resolver una actividad relacionada con el tema abordado. Para finalizar, se presenta un texto argumentativo, con el fin de identificar las partes más importantes e intención comunicativa del discurso argumentativo.

Objetivo de aprendizaje

Identificar la estructura y propósito comunicativo del texto argumentativo.

Duración

2 horas

Modalidad

Individual
