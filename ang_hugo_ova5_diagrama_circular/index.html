Diagrama circular

Introducción

En la primera parte de este recurso encontrará los fundamentos teóricos para la construcción de un diagrama circular. Luego, se plantea un ejemplo sobre cómo realizar de este tipo de diagrama. Por último, se propone una actividad en la que se deberán solucionar cálculos de ángulos y porcentajes para determinar el diagrama circular adecuado. Además, se debe elaborar un análisis de la información suministrada para determinar afirmaciones correctas.

Objetivos de aprendizaje

  • Elaborar diagramas circulares a partir de información proveniente de diferentes fuentes.
  • Interpretar datos representados en diagramas circulares.

Duración

2 horas

Modalidad

Individual
